    Users

    Users are the individuals who participate in the platform. This section covers how to manage and interact with users.

    Register

    URL: /register
    Method: POST
    Request Body:

    {
  "name": "string",
  "email": "string",
  "password": "string" // Must be at least 6 characters long
}

    Response:

    {
  "status": "success",
  "message": "User created",
  "data": {
    "user": {
      "id": "user-123",
      "name": "John Doe",
      "email": "john@example.com",
      "avatar": "https://generated-image-url.jpg"
    }
  }
}

    See All Users

    URL: /users
    Method: GET
    Response:

    {
  "status": "success",
  "message": "ok",
  "data": {
    "users": [
      {
        "id": "john_doe",
        "name": "John Doe",
        "email": "john@example.com",
        "avatar": "https://generated-image-url.jpg"
      },
      {
        "id": "jane_doe",
        "name": "Jane Doe",
        "email": "jane@example.com",
        "avatar": "https://generated-image-url.jpg"
      }
    ]
  }
}

    See Own Profile

    URL: /users/me
    Method: GET
    Headers:

    Authorization: Bearer <token>

    Response:

    {
  "status": "success",
  "message": "ok",
  "data": {
    "user": {
      "id": "john_doe",
      "name": "John Doe",
      "email": "john@example.com",
      "avatar": "https://generated-image-url.jpg"
    }
  }
}
