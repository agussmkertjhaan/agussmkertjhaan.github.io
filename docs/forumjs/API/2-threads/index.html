    Threads

    Threads are discussion topics created by users. This section describes how to create, view, and manage threads.

    Create Thread

    • URL: /threads
    • Method: POST
    • Headers:
    • Authorization: Bearer <token>
    • Request Body:
    • title: string, required. The title of the thread.
    • body: string, required. The content of the thread.
    • category: string (optional). The category of the thread.
    • Response:
    {
  "status": "success",
  "message": "Thread created",
  "data": {
    "thread": {
      "id": "thread-1",
      "title": "First Thread",
      "body": "This is the first thread",
      "category": "General",
      "createdAt": "2024-08-21T07:00:00.000Z",
      "ownerId": "users-1",
      "upVotesBy": [],
      "downVotesBy": [],
      "totalComments": 0
    }
  }
}
    • Additional Information:
    • Authorization: You need to provide a valid Bearer token in the Authorization header.
    • Success Status: "status": "success"
    • Error Status: If the request fails, an appropriate error message will be returned.

    See All Threads

    • URL: /threads
    • Method: GET
    • Response:
    {
  "status": "success",
  "message": "ok",
  "data": {
    "threads": [
      {
        "id": "thread-1",
        "title": "First Thread",
        "body": "This is the first thread",
        "category": "General",
        "createdAt": "2024-08-21T07:00:00.000Z",
        "ownerId": "users-1",
        "upVotesBy": [],
        "downVotesBy": [],
        "totalComments": 0
      },
      {
        "id": "thread-2",
        "title": "Second Thread",
        "body": "This is the second thread",
        "category": "General",
        "createdAt": "2024-08-21T07:00:00.000Z",
        "ownerId": "users-2",
        "upVotesBy": [],
        "downVotesBy": [],
        "totalComments": 0
      }
    ]
  }
}
    • Additional Information:
    • Authorization: Not required for this endpoint.
    • Pagination: If there are many threads, pagination might be needed. This API does not provide pagination for this endpoint.

    See Detail Thread

    • URL: /threads/<threadId>
    • Method: GET
    • Response:
    {
  "status": "success",
  "message": "ok",
  "data": {
    "detailThread": {
      "id": "thread-1",
      "title": "First Thread",
      "body": "This is the first thread",
      "category": "General",
      "createdAt": "2024-08-21T07:00:00.000Z",
      "owner": {
        "id": "users-1",
        "name": "John Doe",
        "avatar": "https://generated-image-url.jpg"
      },
      "upVotesBy": [],
      "downVotesBy": [],
      "comments": [
        {
          "id": "comment-1",
          "content": "This is the first comment",
          "createdAt": "2024-08-21T07:00:00.000Z",
          "owner": {
            "id": "users-1",
            "name": "John Doe",
            "avatar": "https://generated-image-url.jpg"
          },
          "upVotesBy": [],
          "downVotesBy": []
        }
      ]
    }
  }
}

    Additional Information:

    • Authorization: Not required for this endpoint.
    • Comments: Includes comments related to the thread.
    • Error Handling: If the thread ID does not exist, an appropriate error message will be returned.
